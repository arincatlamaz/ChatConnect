# Chat Connect App

The application provides user registration and login, and other users can be searched and chatted with the selected user on the main screen.

## Project Features
• MVVM

• LiveData

• RecyclerView

• Extensions

• Firebase

• Coroutines

• Navigation Component

• DataBinding
